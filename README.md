trakt-tv-status
===============

Node webkit application to view the progress on your TV shows within a desktop application (uses the trakt API and requires your personal api key)

## Requirements

* `npm`
* `bower`


## Installation

```

npm install
bower install
cd app
npm install

```

## Run

### OS X
```
grunt dist-mac && open dist/TraktTvStatus.app/
```
